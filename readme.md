# VCard2GCSV

## Introduction
Utility to convert bunch of vcard files into csv file compatible with Google contacts.

## Dependencies

### Maven
* GNU Getopt 1.0.13
* OpenCSV 2.3
* Apache Commons Codec 1.5 (required by CardMe)

### Included libs
* CardMe 0.4.0

## Usage

`vcard2gcsv -c -i /path/to/input/csv/dir [-o /path/to/output/file.csv]`
